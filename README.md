# Ironridge Hardware Demo

Premium industrial hardware-store website concept built with Next.js, Tailwind CSS and Framer Motion.

## What's included
- Multi-page App Router setup
- Mobile-friendly responsive layout
- Scroll-reactive storytelling section
- Interactive comparison slab
- Clickable command center demo
- Interactive product wall with tilt motion
- Reactive quote builder on the contact page
- Vercel-ready project structure

## Run locally
```bash
npm install
npm run dev
```

## Production check
```bash
npm run build
npm start
```
